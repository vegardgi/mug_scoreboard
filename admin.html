Admin — Scoreboard

For å administrere spillere trenger du et GitHub Personal Access Token med tilgang til dette repoet.

  1. Gå til github.com/settings/tokens/new
  2. Gi tokenet et navn (f.eks. "scoreboard-admin")
  3. Huk av public_repo under "repo"
  4. Klikk "Generate token" og kopier det
